## Project 3 - DecodeLabs AI Internship

This project is a simple AI-based Recommendation System developed using Python. It recommends items based on the category selected by the user.

## 📌 Project Overview

The program contains predefined recommendations for different categories. Users can choose a category, and the system displays relevant recommendations.

### Available Categories

- 🎬 Movies
- 📚 Books
- 🎵 Music
- ⚽ Sports
- 💻 Technology

---

## 🚀 Features

- User-friendly console interface
- Category-based recommendations
- Handles invalid category inputs
- Simple recommendation logic using Python dictionaries
- Beginner-friendly implementation

## 🛠 Technologies Used

## 🧠 Recommendation Logic

The project uses a Python dictionary to store recommendations. Based on the user's selected category:

- If the category exists, related recommendations are displayed.
- If the category does not exist, an appropriate message is shown.
